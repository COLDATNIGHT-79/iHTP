Key Features

Anonymous Posting

No registration required. Users identified only by browser cookies. Complete privacy.

Character Limits

Title: 50 chars | Description: 180 chars | Reason: 300 chars. Concise content only.

Category Tags

Tag posts with up to 2 categories from 6 options including politician, celebrity, athlete.

Like System

Anonymous voting system. Track what you've liked. One like per user per post.

Weekly Leaderboard

See top 5 most-liked posts from the last 7 days. Updates in real-time.

SQLite Database

Free, serverless database. No configuration needed. Everything in one file.
